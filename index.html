Mentor's Note

Recommended learning order

Follow the numbered order: start with Python Basics (the foundation for everything), then Jupyter (your workspace), then NumPy, then spend the most time on Pandas (your core daily tool). After that, Matplotlib & Seaborn for visualisation, and finally Flask if you want to deploy your work on the web.

💡
Practice beats theory

For every concept, type the code yourself — don't just read it. Find a real dataset on Kaggle, apply each concept, and build a small project at the end of each topic.
